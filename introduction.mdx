## What is the Feynman technique?

The Feynman technique is a proven learning method: if you can explain a concept in simple terms, you truly understand it. If you can't, you've found exactly where your knowledge breaks down.

elitecode applies this to DSA practice:

1. **Choose a topic** from the bento grid (Array, Tree, Graph, etc.)
2. **Explain it aloud** — describe what it is, how it works, and when to use it
3. **Get AI feedback** — the AI listens, evaluates your explanation, and tells you whether you've nailed it or where you need to go deeper
4. **Hear the feedback** — the response is spoken back to you using text-to-speech

## Tech stack

elitecode is built with:

- **Next.js 14** — React framework for the frontend
- **OpenAI Whisper** — Speech-to-text transcription (self-hosted via Flask)
- **GPT-3.5-turbo** — AI evaluation of explanations
- **ElevenLabs** — Text-to-speech for spoken AI feedback
- **Framer Motion** — Smooth animations and transitions
- **Tailwind CSS** — Utility-first styling
- **Docker + CUDA** — GPU-accelerated Whisper backend
